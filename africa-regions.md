# Azure Africa Regions Configuration

## Available Africa Regions:
1. **South Africa North** - Johannesburg
2. **South Africa West** - Cape Town

## VM Sizes typically available in Africa regions:
- Standard_B1s (1 vCPU, 1GB RAM)
- Standard_B1ms (1 vCPU, 2GB RAM) 
- Standard_A1_v2 (1 vCPU, 2GB RAM)
- Standard_A2_v2 (2 vCPU, 4GB RAM)

## Pricing Note:
Africa regions might have slightly higher pricing compared to other regions.

## Network Considerations:
- Latency might be higher if accessing from other continents
- Consider using Azure Front Door or CDN for global access
